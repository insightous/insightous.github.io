사용하는 extensions list


검색 창을 이용해 필요한 extension을 설치, 제거, 관리할 수 있다. 현재는 github 아이디를 VS Code와 연동하여 extenstions가 자동 동기화 되는데,
동기화 되지 않을 경우 extensions list를 생성해 다른 pc에서 설치할 수 있다.

  • extensions.list 생성 
    1

    code --list-extensions > extensions.list
  • extensions.list로 설치 
    1

    cat extensions.list |% { code --install-extension $_}
