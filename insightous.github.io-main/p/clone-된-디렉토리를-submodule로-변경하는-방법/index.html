1
2

  git rm --cached .\themes\hugo-theme-stack\
  git submodule add https://github.com/CaiJimmy/hugo-theme-stack/ themes/hugo-theme-stack

  • 즉, clone해온 디렉토리의 git캐시를 지우고, submodule로 추가.

  • .gitmodules 파일을 확인한다.
