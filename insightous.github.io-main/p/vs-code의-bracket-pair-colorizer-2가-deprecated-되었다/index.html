VS Code의 `Bracket Pair Colorizer 2`가 deprecated 되었다.

Jun 28, 2022
1 분 정도

상황


너무나도 잘 사용하고 있던 extension인 Bracket Pair Colorizer 2가 deprecated 되었다.
이 extension은 각 괄호들의 짝을 색으로 구분하여 코드를 읽기 쉽게하는 기능이다.
deprecated 됨에 따라 기능을 대체할 방법을 찾아야 했다.

검색

검색 결과 블로그를 발견했다.
블로그

적용

  • Settings>Workbench>Appearance>Color Customizations의 Edit in settings.json을 눌러 settings.json 파일을 연다.
  • bracket pair coloring 기능 이외에도 Indentation guides colorization 기능도 추가해 준다.
  • Panda Syntax는 테마이름이고 그에 해당하는 색 조합이다. 원하면 다른 테마를 선택해, 그 색조합을 설정하면 된다.
